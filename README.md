# ARNN-KPSS-analysis
0. Dataset for unemployment, US, UK, France, Germany and Italy
1. R-codes for the empirical distribution of KPSS test, Fourier KPSS (FKPSS) test, KPSS with structrual break (KPSS-SB) test, Fourier KPSS with structural break (FKPSS-SB) test and the Autoregressive neural network KPSS (ARNN-KPSS) test
2. R-codes for empirical distribution and graph, KPSS test, FKPSS test, KPSS-SB test, FKPSS-SB test and ARNN-KPSS test
3. R-codes for size and power analysis
4. R-codes for the size and power with neutral network function
5. R-codes for empirical analysis
